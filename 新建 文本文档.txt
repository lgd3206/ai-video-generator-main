● 基于你的AI视频生成器项目代码分析，以下是你需要申请的第三方API接入：

  🎯 核心必需API（优先级最高）

  1. AI视频生成服务

  Replicate API ⭐⭐⭐⭐⭐

  - 用途：文本转视频、图片转视频
  - 申请地址：https://replicate.com
  - 费用：按使用量计费，约$0.1-0.5/分钟视频
  - 环境变量：REPLICATE_API_TOKEN

  RunwayML API （备选）

  - 用途：专业AI视频生成
  - 申请地址：https://runwayml.com/api
  - 费用：订阅制，$12-35/月起
  - 环境变量：RUNWAY_API_KEY

  2. 用户认证服务

  Google OAuth ⭐⭐⭐⭐⭐

  - 用途：用户登录/注册
  - 申请地址：https://console.developers.google.com
  - 费用：免费
  - 环境变量：
    - GOOGLE_CLIENT_ID
    - GOOGLE_CLIENT_SECRET

  3. 数据库服务

  PostgreSQL ⭐⭐⭐⭐⭐

  - 推荐服务：
    - Supabase：https://supabase.com （免费额度大）
    - PlanetScale：https://planetscale.com
    - Neon：https://neon.tech
  - 环境变量：DATABASE_URL

  💰 支付和计费API

  4. Stripe支付 ⭐⭐⭐⭐

  - 用途：订阅付费、信用点购买
  - 申请地址：https://stripe.com
  - 费用：2.9% + $0.30 per transaction
  - 环境变量：
    - STRIPE_SECRET_KEY
    - STRIPE_PUBLISHABLE_KEY
    - STRIPE_WEBHOOK_SECRET

  ☁️ 文件存储服务

  5. 云存储 ⭐⭐⭐⭐

  选择其一：
  Cloudinary（推荐视频处理）

  - 申请地址：https://cloudinary.com
  - 费用：免费额度 25GB/月
  - 环境变量：
    - CLOUDINARY_API_KEY
    - CLOUDINARY_API_SECRET
    - CLOUDINARY_CLOUD_NAME

  AWS S3

  - 申请地址：https://aws.amazon.com/s3/
  - 环境变量：
    - AWS_ACCESS_KEY_ID
    - AWS_SECRET_ACCESS_KEY

  📧 邮件服务API

  6. 邮件发送 ⭐⭐⭐

  Resend（推荐）

  - 申请地址：https://resend.com
  - 费用：免费3000封/月
  - 环境变量：RESEND_API_KEY

  SendGrid（备选）

  - 申请地址：https://sendgrid.com
  - 费用：免费100封/天

  📊 分析和监控API

  7. Google Analytics ⭐⭐⭐

  - 申请地址：https://analytics.google.com
  - 费用：免费
  - 环境变量：NEXT_PUBLIC_GA_ID

  8. Sentry错误监控 ⭐⭐

  - 申请地址：https://sentry.io
  - 费用：免费5000错误/月
  - 环境变量：NEXT_PUBLIC_SENTRY_DSN

  🔧 其他配置

  9. NextAuth密钥 ⭐⭐⭐⭐⭐

  - 生成命令：openssl rand -base64 32
  - 环境变量：NEXTAUTH_SECRET
  - 用途：JWT签名加密

  💡 申请建议和优先级

  第一阶段（MVP）：

  1. Replicate API - 核心视频生成功能
  2. Google OAuth - 用户认证
  3. Supabase - 数据库
  4. NextAuth Secret - 认证加密

  第二阶段（商业化）：

  5. Stripe - 支付功能
  6. Cloudinary - 文件存储
  7. Resend - 邮件通知

  第三阶段（优化）：

  8. Google Analytics - 数据分析
  9. Sentry - 错误监控

  1. 费用控制：设置API使用限额和警报
  2. 安全配置：限制API密钥的域名和权限
  3. 测试环境：为开发和生产创建不同的API密钥
  4. 备份方案：准备备选服务以防主要服务不可用
